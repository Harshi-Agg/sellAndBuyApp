A basic buying and selling application using node.js express and MongoDB with basic CRUD functionalities on user and products

**Functionalities**
1.Add User: New users are created and resgistered to the application.
2.Login: Users with access and account can login to the application.
3.Sell/Add Product: User on dashboard can sell products by adding and creating new products. Products are tagged to some pre-defined categories using relationship in models.
4.Edit Product: Product selling price can be editted.
5.Delete Product: User can delete products.
6.View Products: User can view all the created products.
7.Search products: User can search the products to be bought in search bar using product name.
8.Sort Products: User can sort the products on basis of selling and cost price.

**Models**
1.ProductCategory
2.SellBuy
3.User
